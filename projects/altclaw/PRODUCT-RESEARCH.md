## Validation Process (Per Product)

1. **Find credible review** from:
   - Security professional blogs
   - Tech review sites (Tom's Hardware, Wirecutter)
   - Reddit r/netsec, r/AskNetsec discussions
   - Official Kali Linux documentation

2. **Verify Amazon availability:**
   - Search by product name + specs
   - Confirm ASIN exists
   - Check current price
   - Test affiliate link

3. **Document source:**
   - Link to review/recommendation
   - Quote key benefits
   - Note any limitations

4. **Add to VALIDATED_PRODUCTS.md** with:
   - ASIN
   - Price
   - Commission calculation
   - Review source citation
   - Date validated
