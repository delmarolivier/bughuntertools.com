# Bug Hunter Tools - Deployment

This site is deployed via Netlify with custom domain: bughuntertools.com

## Quick Deploy

1. Push to Git repository
2. Connect to Netlify
3. Deploy automatically

## DNS Configuration

Point domain to Netlify:
- Add A record: 75.2.60.5
- Add CNAME: www -> bughuntertools.netlify.app

## Build Settings

- Build command: (none - static HTML)
- Publish directory: `.`
- No build process required
